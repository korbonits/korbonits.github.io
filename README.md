# korbonits.com

Personal blog and portfolio. Built with [Astro](https://astro.build).

## Development

```bash
npm install
npm run dev
```

## Build & Preview

```bash
npm run build   # builds to dist/
npm run preview # preview the build locally
```

## Deploy

Pushes to `master` automatically deploy to [korbonits.com](https://korbonits.com) via GitHub Actions.
